# HR Employee Attrition Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive **HR Employee Attrition Dashboard** built using **Microsoft Power BI**. The dashboard helps analyze employee attrition trends, workforce demographics, job satisfaction, income distribution, and other HR metrics to identify the key factors influencing employee turnover.

---

## 🚀 Features

- Employee Attrition Analysis
- Department-wise Attrition
- Job Role-wise Attrition
- Age Group Analysis
- Gender-wise Attrition
- Overtime Impact on Attrition
- Job Satisfaction Analysis
- Interactive Filters (Slicers)
- KPI Cards for Quick Insights

---

## 📊 Dashboard KPIs

- Total Employees
- Employees Left
- Attrition Rate
- Average Monthly Income

---

## 📈 Visualizations

- Department-wise Attrition (Bar Chart)
- Job Role-wise Attrition (Bar Chart)
- Age Group Analysis (Column Chart)
- Gender Distribution (Pie Chart)
- Overtime vs Attrition (Bar Chart)
- Job Satisfaction Analysis (Column Chart)

---

## 🎛 Interactive Filters

- Department
- Gender
- Age Group
- Job Role
- Business Travel

---

## 🧹 Data Cleaning

The dataset was cleaned using **Power Query** by:

- Removing unnecessary columns
- Checking data types
- Handling missing values
- Removing duplicate records
- Creating Age Group categories

---

## 🛠 Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)

---

## 📂 Dataset

IBM HR Analytics Employee Attrition & Performance Dataset

---

## 💡 Key Insights

- Identified departments with the highest employee attrition.
- Analyzed the impact of overtime on employee turnover.
- Compared attrition across different age groups.
- Studied job satisfaction levels and their relationship with attrition.
- Examined employee distribution across job roles.
